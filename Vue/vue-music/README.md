# vue-music

## 实现的功能：
1. 排行榜
2. 轮播图展示
3. 搜索推荐
4. 搜索歌曲
5. 搜索专辑
6. 搜索歌手
7. 上一曲，下一曲，播放暂停
8. 随机播放，循环播放，单曲播放
9. 加入/删除播放列表，
10. 查看专辑
11. 查看歌手
12. 登录
13. 注册
14. 收藏
